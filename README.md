## Status
Em desenvolvimento

## Tecnologias

- Swing
- MySQL

## Desenvolvedores

## Objetivo
Aplicativo desktop para criar, armazenar e revisar flashcards.

## Funcionalidades
- Cadastrar flashcards
- Consultar flashcards existentes
- Navegar entre flashcards
- Virar flashcards para ver respostas
